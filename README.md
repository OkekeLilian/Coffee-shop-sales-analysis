### ☕ Coffee Shop Sales Analysis 

##Project Overview
Analysed 149,116 transactions from a coffee shop chain across 3 new york locations (jan-jun 2023) using microsoft excel

##Tools used 
-  Microsoft Excel
- Pivot Tables
- VLOOKUP, HOUR, TEXT functions
- Conditional Formatting
- Data Visualization (Charts & Dashboard)

## Key Insights
- Coffee is the top revenue category generating $269,952 which is 38% of total sales  
- Peak trading hour is 10:00–10:59 with 26,713  transactions across all stores
- Revenue grew 104% from January ($81,677)  to June ($166,485)
- All 3 locations perform almost equally at roughly 33% revenue share each


 ## Dataset
Source: Maven Analytics — Coffee Shop Sales
Period: January to June 2023
Rows: 149,116 transactions

## Workbook Structure
- Raw Data — original unmodified dataset
- Cleaned Data — added Revenue, Month, Day of  Week, Hour and Time of Day columns 
- Pivot Tables — summary tables for analysis
- Dashboard — visual summary of key findings
- Formulas Used — reference sheet of all functions used
